Money-Flow Screener · User Guide

使い方ガイド

このダッシュボードが何を見せているのか、各パネルの読み方、用語の意味をまとめています。

1. このツールの目的

日本株 __N__銘柄（TOPIX500＋日経225＋話題枠）の売買代金を1分足から集計し、 「いま、どこに資金が集まりだしているか」を可視化するためのダッシュボードです。

株価そのものの上下ではなく、売買代金（終値×出来高）の集中に注目します。 価格が動く前に出来高が先に膨らむことは多く、「資金の向き先」を早めに捉えるのが狙いです。 真の板情報（気配値）や歩み値のリアルタイムは無料の範囲では取得できないため、 売買代金の集中で近似している点にご留意ください。

画面右上には 「◯◯ 銘柄」（現在の監視総数）と 「話題枠 ＋N」 （指数母数〈TOPIX500＋日経225〉に上乗せされた話題枠の銘柄数）を表示します。 話題枠は毎週入れ替わるので、この数は変動します。

2. データの流れと更新頻度

Yahoo Finance（1分足）→ 約5分ごと収集（場中）→ 資金集中スクリーニング→ 自動デプロイ
  • 東証の立会時間（平日9:00〜15:30）に約5分ごと収集・再計算し、ページが自動更新されます。 アラートが出るまでの遅れはおよそ5〜8分です（Yahoo自体の1〜2分遅れ＋収集＋デプロイ）。
  • 1分足は直近およそ7営業日分を保持（それより前は月次アーカイブへ退避）。
  • 日足・週足・月足は引け後に1回更新（日足2年分から自前で集約）。
  • 話題枠は毎週月曜の寄り前に自動で入れ替わります（下記「今週の話題枠 入れ替え」）。
  • 更新はGitHub Actions＋外部cronで自動化。休場日・昼休みに走っても新規バーが無いだけです。
  • 秒単位の真のリアルタイムや板情報（気配値）は無料の範囲では扱えないため、 売買代金の集中で資金の向きを近似しています。

3. 各パネルの読み方

集計窓 / 急騰しきい値（画面上部の切替）

集計窓は「どのくらいの時間幅で資金の集中を見るか」の選択です（1分〜4時間、日足・週足・月足）。 すべてのパネルがこの窓に連動します。急騰しきい値は、各銘柄のユニバース内シェア （全__N__銘柄の売買代金に占める割合）が直前の同じ窓と比べて何pp増えたら「急騰🚨」とみなすかの感度です （+0.2＝敏感／+0.5＝標準／+1.0＝厳選）。

🧭 自動分析コメント

収集のたびに、算出できた事実だけを日本語でまとめます。主な行は次のとおりです：

  • 全体活況度：直近窓の売買代金が平常時（履歴中央値）の何倍か（過熱／閑散）。
  • 業種別シェア／業種別の資金移動：どの業種が資金を集めているか、どこへ流入・どこから流出したか。
  • 資金流入／流出の上位銘柄：シェアが拡大／縮小した銘柄（株価の騰落も併記）。
  • シェア急伸と株価の連動：シェアが急伸した銘柄のうち、株価も伴うものと「出来高先行」の件数。
  • 大口空売り報告：JPXの大口空売り残高と資金集中が重なっている銘柄（事実のみ）。

ニュースの理由や反転予測などの推測は書きません（データで裏づけられないため）。 各行の先頭ラベルは太字、増減の数値は緑（+）／赤（−）で色分けしています。

どの層に資金が寄っているか

銘柄を3つの層に分けたシェアです。けん引大型＝日経225、主力＝TOPIX500、 話題・噂＝話題枠（毎週月曜に自動入替、下記参照）。

業種別シェア（JPX33業種＋半導体）

  • 積み上げバー：上位8業種＋その他のシェアをひと目で。
  • 勢力図マップ：全区分を敷き詰めた市場マップ。タイルの面積＝シェア、 色＝通常時からのシェア変化（緑＝シェアを奪っている／ 赤＝奪われている）。タイルにカーソルを合わせると、 その業種の資金流入トップ3銘柄まで表示します。
  • 区分はJPX公式33業種に、公式には無い独自区分「半導体」（電気機器等に散っている半導体主力13銘柄を切り出し）を加えたものです。
  • 極小の業種はタイルが潰れないよう下限サイズで表示するため、面積は厳密なシェア比例ではありません。正確な数値はマップ下の一覧で確認できます。

資金集中ランキング

集中度＝直近窓の売買代金 ÷ 平常時（履歴の同じ窓幅の中央値）。 「いつもの何倍の資金が集まっているか」を表します。これは集中度が高い上位18銘柄の絞り込みリストで、 どの銘柄も資金が集中したときにここへ自動で現れます。行をクリックするとその銘柄のチャートに切り替わります。

🔁 今週の話題枠 入れ替え

話題枠（下記）は毎週月曜の寄り前に自動で入れ替わり、その週の追加／除外を根拠付きで表示します。

  • 追加：Yahoo Finance の出来高・値上がり率ランキング上位に登場した、監視ユニバース外の個別株 （ETF・投信などは除外）。「どのランキングの何位か」を根拠として表示します。
  • 除外：4週連続でランキング圏外かつ直近の集中度が平常（1.3倍未満）になった古い話題株。 「何週圏外・集中度何倍か」を根拠として表示します。
  • TOPIX500・日経225の銘柄は絶対に外しません。半導体などの恒久テーマ銘柄と、今まさに集中している銘柄も保護します。

🔥 話題枠 一覧

話題枠の全銘柄を集中度順に常時一覧します（集中度・直近売買代金・勢いΔpp・期間騰落・空売り残）。 行をクリックするとチャートが開きます。追加されたばかりの銘柄の業種欄は「—」で、翌月の月次更新でJPX業種が付きます。

銘柄チャート / 全銘柄を表で見る

チャートは足（1分〜月足）を切替可能（1分のみ終値ライン、他はローソク足）。 「全銘柄を表で見る」を開くと、ランキング外を含む全__N__銘柄が並び、 どの行をクリックしてもその銘柄のチャートが開きます。

空売り残バッジ

JPX公式「空売りの残高に関する情報」（発行済株式数0.5%以上を報告した大口投資家のみ・日次）です。 表示が無い銘柄は「空売りが無い」のではなく「大口報告が無い（基準未満または未報告）」という意味です。

4. 用語集

シェア
全__N__銘柄の売買代金に占める、その銘柄／業種の割合（%）。
Δpp（デルタpp）
シェアの差（パーセントポイント）。通常時や直前の窓と比べてどれだけ増減したか。
集中度（surge）
直近窓の売買代金 ÷ 平常時（履歴中央値）。1.0で平常並み、大きいほど資金が集中。
勢いΔpp（momentum）
直前の同じ窓と比べたユニバース内シェアの増減。急騰しきい値超で🚨。
通常時 / 平常時
履歴を同じ窓幅で区切ったときの中央値。その銘柄・業種の「いつも」の水準。
出来高先行
売買代金（シェア）は増えているのに株価がまだ追随していない状態の事実描写。
話題枠（hot）
TOPIX500・日経225に含まれない「今話題の個別株」。出来高・値上がり率ランキングから毎週月曜に自動入替。
けん引大型／主力
けん引大型＝日経225、主力＝TOPIX500（話題枠を除く）。

5. 制約と免責

  • 非公式・無認証APIを利用しているため、遅延や一時的な欠測が起こり得ます。
  • 「急騰」は売買代金集中という事実の検知であり、値上がりの予測でも売買の推奨でもありません。
  • 空売り残高は大口報告分のみで、市場全体の空売り比率・信用残は含みません。
  • 話題枠の自動選定は外部ランキング由来のため、取得できない週は前週の話題枠を維持します。
  • 板情報・歩み値のリアルタイムは扱っていません（売買代金の集中で近似）。
  • 本ツールは事実の可視化であり、投資助言ではありません。投資判断はご自身の責任で行ってください。
