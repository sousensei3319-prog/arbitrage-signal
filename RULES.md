# トレードルール統合ガイド (ロング & ショート)

このリポジトリは2本のシグナルツールで構成される。**FGI(恐怖と欲望指数)で使い分ける**のが大原則。

| 市場心理 | 使うツール | 狙い |
|---|---|---|
| FGI **高い (>70 強欲)** | `unified_signal.py` | ショート (ロング過熱の逆噴射) |
| FGI **低い (<30 恐怖)** | `long_signal.py` | ロング (売られすぎの反発) |
| 中立 (30〜70) | 両方 | 個別銘柄の歪み次第で両建て判断 |

---

## ショート: unified_signal.py

3層フィルタ (マクロ→ポンプ検知→歪みスコア) で「ロングが過熱した銘柄」を炙り出す。

- Layer1: FGI + BTC方向 (BTCラリー中はショート控え)
- Layer2: 急騰 + 過去のダンプ再現性
- Layer3: FRプラス過熱 + ベーシスプレミアム + OI急増 + L/S過熱
- **エントリー**: ポンプ後の下落確定 (1h終値が直近安値割れ)
- **利確**: 半値戻し → 全戻し / **損切**: 当日高値+2%

## ロング: long_signal.py — 3戦略

マクロ環境(FGI+BTC)で各戦略を自動ON/OFF。

### A. 踏み上げ逆張り (Short Squeeze Reversal) 🔄
- **発動条件(自動)**: FGI≤55 または BTC急落後
- **点灯条件**: FRマイナス過熱 + L/Sショート偏り(<1.0) + 7日高値から暴落
- **理屈**: ショートが過剰 → きっかけで買い戻し殺到 → 自己増殖的に急騰
- **エントリー**: 下げ止まり確認 (1h終値が直近6h高値を上抜け)
- **利確**: TP1=暴落の半値戻し / TP2=全戻し(7日高値)
- **損切**: 7日安値-1%
- **罠**: 落ちるナイフ。FRマイナスでも下げ続ける銘柄あり。必ず下げ止まり確認

### B. モメンタム順張り (Breakout Momentum) 🚀
- **発動条件(自動)**: BTC上昇トレンド時のみ (地合い必須)
- **点灯条件**: 初動(1h+3〜12%) + OI増加 + 出来高急増 + 4hEMA上
- **理屈**: 上がり始めに乗ってトレンドの伸びを取る
- **エントリー**: レジスタンス上抜けの1h終値確定
- **利確**: TP1/TP2 (値幅の0.5倍/1.0倍) or トレイリング
- **損切**: 直近3h安値割れ (浅く速く)
- **罠**: 勝率30-40%と低い。ダマシ多い。損切り厳守。手数料負け注意

### C. マクロスイング (Macro Swing) 💎 ★統計最強★
- **発動条件(自動)**: FGI≤30 (≤20で強発動)
- **理屈**: 機関の買い(ETFフロー)と市場の底(FGI)に相乗り
- **対象**: BTC/ETH/主要アルト (個別小型でなく地合いを買う)
- **エントリー**: 恐怖の底で分割買い(DCA)
- **利確**: FGIが強欲(70+)に戻ったら段階利確
- **損切**: スイング安値割れ(広め)
- **統計**: FGI<25で7年+1,145% / FGI<10は過去6年の90日平均+48%・マイナス事例ゼロ
- **必須確認**: ETFフロー (https://sosovalue.com/ja/assets/etf/us-btc-spot) が連続プラスか

## ④ スマートマネー追跡の運用ルール (2026-07-03 追加)

HL勝ち組ウォレットの追跡シグナル (`smart_money_tracker.py` / 週次レポート)。
30日イベントスタディで **+24h平均+1.4%・勝率60%** (ベースライン±0.2%を超過, n=51)。

### 通知が来たときの手順

1. **コンセンサス新規参入 (🐋)**: 3人以上が同一銘柄・同方向。
   添付チャートで勝ち組の平均entry価格を確認 → **今の価格が彼らより不利なら見送り**。
   基本保有 24〜72h (検証で+24h以降もエッジ持続)。
2. **VIPムーブ (⭐)**: 実現PnL上位8人の単独大口 ($50k+)。1人の判断なので
   コンセンサスより弱いシグナル。方向の参考+監視リスト入りに留める。
   VIPの**クローズ通知**は「彼らの利食い/撤退」— 自分も持っていたら再点検。
3. **必ずFGI/マクロと合流**: LONGシグナル×FGI恐怖圏 or FRマイナス過熱 = 最強。
   ショート地合い (FGI>70) でのLONGコンセンサスは片目で見る。
4. **拒否権フィルターは自動適用済み**: unified_signal/long_signalは
   スマートマネーが$2M以上逆向きの銘柄を自動で落とす (ログに🛑表示)。

### 禁止事項

- シグナル即エントリー (遅行データ。彼らはあなたより良い価格で入っている)
- Botウォレットの執行のコピー (レイテンシで必敗。追跡から自動除外済み)
- 閾値をいじって通知を増やすこと (多すぎる通知は無視される。§5スイープで検証してから)

### メンテナンス

- **月1回**: Actions → Smart Money Collect 実行 (追跡リスト/VIPの入れ替わり反映)
- 発火履歴 (`smart_money_signals_log.csv` / `smart_money_vip_log.csv`) を
  四半期ごとに突き合わせ、勝率が落ちていたら閾値・追跡母数を見直す
